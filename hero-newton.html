定律·牛顿力学系

掌握牛顿三大定律，理解力与运动的关系。这是高中物理的核心，连接了受力分析和运动学，是解决动力学问题的金钥匙！

核心知识点

1️⃣牛顿第一定律（惯性定律）

内容：一切物体总保持匀速直线运动状态或静止状态，直到有外力迫使它改变这种状态为止。

关键理解：

  • 揭示了力的本质：力是改变物体运动状态的原因，不是维持运动的原因
  • 定义了惯性：物体保持原有运动状态的性质
  • 惯性大小只与质量有关，质量越大惯性越大

💡 惯性是物体的固有属性，与运动状态无关！

2️⃣牛顿第二定律（F=ma）

内容：物体的加速度与所受合外力成正比，与质量成反比，加速度方向与合外力方向相同。

F = ma

理解要点：

  • F是合外力，不是某一个力
  • a与F同向，瞬时对应关系
  • 矢量式：Fₓ = maₓ，Fᵧ = maᵧ

两种表达形式：

  • 已知力求运动：先求a，再求v、x
  • 已知运动求力：先求a，再求F
3️⃣牛顿第三定律（作用力与反作用力）

内容：两个物体之间的作用力和反作用力总是大小相等、方向相反、作用在同一条直线上。

特点：

  • 大小相等、方向相反、共线
  • 作用在两个不同的物体上（不能相互抵消）
  • 同时产生、同时消失
  • 性质相同（都是弹力、或都是摩擦力等）

⚠️ 注意：作用力与反作用力 ≠ 平衡力！

⚖️作用力与反作用力 vs 平衡力
比较项作用力与反作用力平衡力
作用物体两个不同物体同一物体
力的性质相同可以不同
同时性同时产生、消失不一定
效果不能抵消可以抵消（平衡）
📐牛顿第二定律解题步骤

第一步：明确研究对象（单个物体或系统）

第二步：受力分析画出受力图

第三步：建立坐标系通常沿加速度方向为x轴

第四步：列方程

  • Fx = ma（沿加速度方向）
  • Fy = 0（垂直加速度方向，平衡）

第五步：求解并检验

🎮游戏联想
  • 第一定律：鲁班的炮弹不受力就直线飞行（惯性）
  • 第二定律：F=ma，装备越好（质量大）越难改变方向，加速需要更多力
  • 第三定律：你打敌人多疼，反作用力让你手多疼（作用反作用）

经典例题

📝例题1：瞬时性问题

题目：如图所示，质量为m的小球用水平弹簧和倾角为30°的光滑木板AB托住，处于静止状态。弹簧被拉伸，弹力大小为mg/2。当木板AB突然向下撤离的瞬间，小球的加速度为（　）

解析：

关键：分析撤去木板前后的受力变化

撤去前（静止）：

  • 重力 mg（向下）
  • 弹簧弹力 F = mg/2（水平向右）
  • 木板支持力 N（垂直木板向上）

撤去后（瞬时）：

  • 弹簧弹力不变（弹簧形变来不及改变）F = mg/2
  • 重力 mg
  • 木板支持力变为0

求合外力：

水平方向：Fx = F = mg/2

竖直方向：Fy = mg

合力 F合 = √[(mg/2)² + (mg)²] = √(5mg²/4) = (√5/2)mg

加速度 a = F合/m = (√5/2)g

✅ 答案：a = (√5/2)g

💡 弹簧弹力不能突变，绳的拉力可以突变！

📝例题2：连接体问题

题目：如图所示，质量为M的木板放在光滑水平面上，木板上表面粗糙。质量为m的小木块以初速度v₀从木板左端滑上木板，求木块和木板的加速度。

解析：

分析摩擦力方向：

  • 木块m相对木板向右运动
  • m受到的滑动摩擦力向左（阻碍相对运动）
  • 木板M受到的滑动摩擦力向右（作用力与反作用力）

设滑动摩擦因数为μ：

对木块m：f = μmg = ma₁

∴ a₁ = μg（方向向左，减速）

对木板M：f = μmg = Ma₂

∴ a₂ = μmg/M（方向向右，加速）

✅ 答案：木块加速度a₁=μg（向左），木板加速度a₂=μmg/M（向右）

📝例题3：超重与失重

题目：一个人站在体重计上，突然下蹲，体重计的示数如何变化？

解析：分析下蹲过程的运动状态：

阶段一：加速下蹲（速度向下且增大）

  • 加速度向下，处于失重状态
  • N = mg - ma < mg，示数变小

阶段二：减速下蹲（速度向下且减小）

  • 加速度向上，处于超重状态
  • N = mg + ma > mg，示数变大

结论：先变小（失重），后变大（超重），最后恢复mg

✅ 答案：示数先变小后变大

💡 口诀：上超下失（加速度向上超重，向下失重）
