核心知识点
🧭 磁场与磁感线
  • 磁场：磁体或电流周围存在的一种特殊物质
  • 磁感线：描述磁场分布的假想曲线
  • 方向：磁感线上某点的切线方向表示该点磁场方向
  • 特点：磁感线是闭合曲线，不相交、不相切
⚡ 安培力

通电导线在磁场中受到的力

F = BIL·sinθ

当B⊥I时（θ=90°）：F = BIL

✋

左手定则：

磁感线穿掌心，四指电流方向，拇指受力方向

⚡ 洛伦兹力

运动电荷在磁场中受到的力

f = qvB·sinθ

当v⊥B时（θ=90°）：f = qvB

  • 方向：左手定则（四指正电荷运动方向）
  • 特点：洛伦兹力永不做功（始终⊥v）
🔄 带电粒子在匀强磁场中的运动

当v⊥B时，带电粒子做匀速圆周运动

洛伦兹力提供向心力：qvB = mv²/r
半径：r = mv/(qB)
周期：T = 2πm/(qB)
🎮 王者技巧

记住口诀："半径看速度，周期看比荷"。周期与速度无关！

🔬 质谱仪原理

构造：加速电场 + 偏转磁场

原理：

  1. 电场加速：qU = ½mv² → v = √(2qU/m)
  2. 磁场偏转：qvB = mv²/r → r = mv/(qB)
  3. 联立得：m/q = B²r²/(2U)

用途：测定带电粒子的比荷(q/m)或质量，分离同位素

⚡ 回旋加速器原理

构造：两个D形盒 + 交变电压

原理：

  • 电场加速：粒子在两D形盒间隙被加速
  • 磁场偏转：在D形盒内做半圆运动
  • 同步条件：电场周期 = 粒子回旋周期
T电场 = T粒子 = 2πm/(qB)
💡 最大动能：由D形盒半径R决定，Ekmax = (qBR)²/(2m)

经典例题
📝 例题1：安培力计算

如图所示，一段长L=0.5m的通电导线，电流I=2A，放在磁感应强度B=0.4T的匀强磁场中，电流方向与磁场方向垂直。求导线受到的安培力大小和方向。

解：

因电流方向与磁场方向垂直，θ=90°，sinθ=1

根据安培力公式：F = BIL

F = 0.4 × 2 × 0.5 = 0.4 N

方向由左手定则判断（垂直于电流和磁场所在平面）

📝 例题2：带电粒子在磁场中的圆周运动

一个带电粒子（电荷量q，质量m）以速度v垂直射入磁感应强度为B的匀强磁场中，测得粒子做圆周运动的半径为R。求：(1)粒子的速度v；(2)运动周期T。

解：

(1) 洛伦兹力提供向心力：qvB = mv²/R

解得：v = qBR/m

(2) 周期公式：T = 2πR/v

代入v：T = 2πR/(qBR/m) = 2πm/(qB)

💡 物理本质

周期与速度无关，这是回旋加速器能工作的关键原理！

📝 例题3：质谱仪问题

某种质谱仪的工作原理如图所示。带电粒子（电荷量均为+q）先经电压为U的加速电场加速，然后进入磁感应强度为B的匀强磁场做匀速圆周运动，最后打在照相底片上的P点。已知粒子质量为m，求OP的距离。

解：

第一步：电场加速

由动能定理：qU = ½mv²

v = √(2qU/m)

第二步：磁场偏转

qvB = mv²/r

r = mv/(qB) = m/(qB) × √(2qU/m) = √(2mU/q)/B

第三步：求OP

OP = 2r = 2√(2mU/q)/B
