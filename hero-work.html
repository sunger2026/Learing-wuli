核心知识点

💪 功的定义 Work

力与物体在力的方向上移动距离的乘积叫做功。

W = F · s · cosθ = F · x · cosθ

F = 力的大小（单位：牛顿 N）

s (或x) = 位移（单位：米 m）

θ = 力与位移方向的夹角

W = 功（单位：焦耳 J）

💡 关键理解：

  • 功是标量，只有大小没有方向
  • 当力与位移垂直时(θ=90°)，不做功(cos90°=0)
  • 当力与位移同向时(θ=0°)，W = Fs（最大）
📊 正功与负功

根据力与位移的夹角判断做功的正负：

0° ≤ θ < 90° → W > 0（正功）
θ = 90° → W = 0（不做功）
90° < θ ≤ 180° → W < 0（负功）

💡 物理意义：

  • 正功：力促进物体运动，是动力
  • 负功：力阻碍物体运动，是阻力
  • 不做功：力对物体运动无影响

⚠️ 常见误区：摩擦力一定做负功？不一定！如传送带带动物体时，静摩擦力做正功。

⚡ 功率 Power

单位时间内完成的功，表示做功的快慢。

P = W/t = F·v·cosθ

P = 功率（单位：瓦特 W）

W = 功（单位：焦耳 J）

t = 时间（单位：秒 s）

F = 牵引力，v = 速度

两种功率：

  • 平均功率：P̄ = W/t = F·v̄·cosθ
  • 瞬时功率：P = F·v·cosθ（某一时刻的功率）
🚗 机车恒定功率启动

汽车以恒定功率启动时的运动特点：

P = F·v（恒定）
F - f = ma

运动过程分析：

  • 启动时v小，F = P/v大，加速度a大
  • 速度增大，牵引力F减小，加速度a减小
  • 当F = f时，a = 0，达到最大速度vₘ
  • 之后匀速运动：vₘ = P/f

🚀 机车恒定加速度启动

汽车以恒定加速度启动时的运动特点：

F - f = ma（恒定）
P = F·v

运动过程分析（两个阶段）：

  • 第一阶段（匀加速）：
    • F恒定，a恒定，v均匀增加
    • P = Fv 逐渐增大
    • 直到P达到额定功率P₀
  • 第二阶段（变加速）：
    • P = P₀ 保持恒定
    • v增大，F减小，a减小
    • 直到F = f，达到最大速度

📐 功的计算方法总结

计算功的几种方法：

  • 定义法：W = Fscosθ（恒力做功）
  • 功率法：W = Pt（功率恒定或已知平均功率）
  • 动能定理：W合 = ΔEₖ = ½mv₂² - ½mv₁²
  • 图像法：F-s图像中，图线与横轴围成的面积表示功
  • 转换法：通过能量转化关系间接求功

💡 选择技巧：

  • 恒力做功直接用定义式
  • 变力做功优先考虑动能定理
  • 机车问题优先考虑功率公式

🗡️ 用王者荣耀理解功和功率
  • 后羿射箭：拉弓的力F，箭移动距离s，做功W = Fs。拉得越满，做功越多，箭速越快！
  • 亚瑟冲锋：大功率输出！短时间内做大量功，爆发伤害高。P = W/t，技能冷却短=功率大。
  • 关羽骑马：持续做功的典范。马力（功率）恒定，速度越快，牵引力越小。
  • 韩信跳跃：起跳时肌肉做功W = mgh，将化学能转化为重力势能，跳得越高做功越多。
⚡ 功率实战应用
  • 打野效率：打野怪的速度就是功率！P = 伤害/时间，高功率=快速清野。
  • 推塔速度：攻击频率 × 每次伤害 = 推塔功率，功率越大推塔越快。
  • 技能连招：一套技能的总伤害/释放时间 = 技能功率，优化连招顺序提高功率输出。

经典例题

📝 例题1：功的计算

题目：

一个质量为10kg的物体，在水平地面上受到与水平方向成37°角的拉力F = 50N作用，

沿水平面移动了10m。已知sin37°=0.6，cos37°=0.8。

(1) 求拉力F做的功。

(2) 若物体匀速运动，求摩擦力做的功。

解析：

(1) 拉力做的功：

WF = F · s · cosθ = 50 × 10 × 0.8 = 400 J

(2) 摩擦力做的功：

物体匀速运动，水平方向合力为零

f = Fcos37° = 50 × 0.8 = 40 N

摩擦力与位移方向相反，做负功

Wf = -f · s = -40 × 10 = -400 J

✅ 答案：(1) 400 J；(2) -400 J

💡 注意：摩擦力做负功，也可以说成"物体克服摩擦力做功400J"

📝 例题2：功率计算

题目：

一辆汽车的质量为2×10³kg，发动机的额定功率为80kW。

汽车在平直公路上行驶时所受阻力恒为4000N。

(1) 求汽车行驶的最大速度。

(2) 若汽车以2m/s²的加速度匀加速启动，求维持匀加速运动的最长时间。

解析：

(1) 最大速度：

当汽车达到最大速度时，牵引力F = 阻力f

P = F·vm = f·vm

vm = P/f = 80000/4000 = 20 m/s

(2) 匀加速最长时间：

匀加速阶段：F - f = ma

F = f + ma = 4000 + 2000×2 = 8000 N

匀加速结束时的速度：v = P/F = 80000/8000 = 10 m/s

由v = at，得 t = v/a = 10/2 = 5 s

✅ 答案：(1) 20 m/s；(2) 5 s

📝 例题3：机车启动综合问题

题目：

一列火车总质量为500t，机车额定功率为600kW，轨道阻力为车重的0.01倍。

火车从静止开始以额定功率启动（g=10m/s²）。

(1) 当速度为10m/s时，火车的加速度是多少？

(2) 火车的最大速度是多少？

解析：

首先计算阻力：f = 0.01mg = 0.01 × 500×10³ × 10 = 5×10⁴ N

(1) v = 10m/s时的加速度：

牵引力 F = P/v = 600×10³/10 = 6×10⁴ N

由牛顿第二定律：F - f = ma

a = (F-f)/m = (6×10⁴ - 5×10⁴)/(500×10³) = 0.02 m/s²

(2) 最大速度：

当F = f时，速度达到最大

vm = P/f = 600×10³/(5×10⁴) = 12 m/s

✅ 答案：(1) 0.02 m/s²；(2) 12 m/s

🎮 游戏联想：就像关羽骑马，速度越快，牵引力越小，加速越慢！
