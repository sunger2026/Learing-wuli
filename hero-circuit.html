核心知识点
🔋 电源与电动势
  • 电源：将其他形式能转化为电能的装置
  • 电动势(E)：反映电源将其他能转化为电能的本领
  • 内阻(r)：电源内部的电阻
  • 路端电压：电源两极间的电压
⚡ 闭合电路欧姆定律
I = E / (R + r)

或：E = U外 + U内 = IR + Ir

路端电压：U = E - Ir
🎮 王者技巧

外电阻R增大→电流I减小→内电压Ir减小→路端电压U增大。记住：断路时(R→∞)，U = E！

🔀 串并联电路

串联电路：

  • 电流处处相等：I = I₁ = I₂ = ...
  • 总电压等于各部分电压之和：U = U₁ + U₂ + ...
  • 总电阻：R = R₁ + R₂ + ...

并联电路：

  • 各支路电压相等：U = U₁ = U₂ = ...
  • 总电流等于各支路电流之和：I = I₁ + I₂ + ...
  • 总电阻：1/R = 1/R₁ + 1/R₂ + ...
⚡ 电功率与焦耳定律

电功率：

P = UI = I²R = U²/R

焦耳定律（热功率）：

Q = I²Rt
💡 区分：纯电阻电路：W = Q；非纯电阻电路（如电动机）：W > Q，电能部分转化为机械能。
📊 U-I图像分析

电源U-I图像：

  • 纵轴截距 = 电动势E（断路电压）
  • 横轴截距 = 短路电流 I短 = E/r
  • 斜率绝对值 = 内阻r

电阻U-I图像：过原点的直线，斜率=R

交点含义：两图像交点表示该电阻接在该电源上的工作状态

🔬 电阻测量方法

伏安法：

  • 内接法：电流表内接，适合测大电阻（R >> RA）
  • 外接法：电流表外接，适合测小电阻（R << RV）

口诀："大内偏大，小外偏小"

经典例题
📝 例题1：闭合电路计算

如图所示电路，电源电动势E=12V，内阻r=1Ω，外电阻R=5Ω。求：(1)电路中的电流；(2)路端电压；(3)电源输出功率。

解：

(1) 根据闭合电路欧姆定律：

I = E/(R+r) = 12/(5+1) = 2 A

(2) 路端电压：

U = IR = 2×5 = 10 V

或 U = E-Ir = 12-2×1 = 10 V

(3) 电源输出功率：

P出 = UI = 10×2 = 20 W

或 P出 = I²R = 4×5 = 20 W

📝 例题2：动态电路分析

如图所示电路，电源电动势和内阻恒定，当滑动变阻器滑片P向右移动时，分析各电表示数如何变化？

分析思路（串反并同）：

滑片P右移→滑动变阻器阻值Rₚ增大

总电路：R总增大→I总减小（电流表示数减小）

路端电压：U = E-Ir，I减小→U增大（电压表示数增大）

定值电阻R₁：与变阻器串联，电流减小→U₁=IR₁减小

并联部分：U并 = U-U₁，U增大、U₁减小→U并增大→I₂增大

🎮 口诀记忆

"串反并同"：与变化电阻串联的，变化相反；并联的，变化相同。

📝 例题3：最大功率问题

电源电动势E，内阻r。外电路为纯电阻R时，R为多大时电源输出功率最大？最大功率是多少？

解：

输出功率 P = I²R = [E/(R+r)]²·R = E²R/(R+r)²

利用数学知识：当R = r时，分母最小，P最大。

最大功率：

Pmax = E²/(4r)
💡 结论：当外电阻等于内阻时，电源输出功率最大，此时效率只有50%。
