## 🎯 项目背景

针对**湖南高一学生**（初中物理基础薄弱）开发的游戏化物理学习网页，目标是：
- 📚 系统梳理初中+高中全部物理知识
- 🎮 用游戏化设计提升学习兴趣
- 📝 专项训练湖南高考真题
- 📈 通过段位系统激励持续学习

## ✨ 核心功能

### 🦸 英雄卡系统
每个物理知识点都化身为一位"英雄"，学生通过"召唤英雄"来学习：

| 模块 | 英雄数量 | 覆盖内容 |
|------|---------|---------|
| 🎮 新手村 | 12位 | 初中物理全部知识点 |
| 📗 必修一 | 6位 | 高一上学期（运动学、力学） |
| 📘 必修二 | 6位 | 高一下学期（曲线运动、能量） |
| 📙 选修模块 | 10位 | 电场、磁场、电磁感应、动量、热学、光学、原子物理 |
| 🔬 实验专题 | 1位 | 高考必考实验 |

### ⚔️ 对战模式
- **每日首胜** - 每天5道题，连胜有奖励
- **排位赛** - 挑战更高段位，证明实力
- **BOSS战** - 综合大题，击败拿经验
- **错题本** - 消灭错题，强化弱点

### 📝 湖南高考真题专区
- 📅 2022-2024年湖南高考物理真题
- 🎯 题型专练：选择题 / 实验题 / 计算题 / 选考题
- 📊 详细解析与考点分析

### 🏆 段位系统
```
青铜 → 白银 → 黄金 → 铂金 → 钻石 → 星耀 → 王者
```

## 🚀 快速开始

### 在线访问
直接访问部署好的网站：
- 🌐 [物理王者主页](http://你的服务器地址/)

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/你的用户名/physics-king.git

# 进入目录
cd physics-king

# 启动本地服务器
python3 -m http.server 8080

# 浏览器访问
open http://localhost:8080
```

## 📚 知识点覆盖

### 初中物理（新手村）
- 机械运动、声现象、光现象
- 物态变化、质量与密度
- 力与运动、压强、浮力
- 简单机械、电学基础、电功率、磁学

### 高中物理
- **必修一**：运动的描述、匀变速直线运动、自由落体、相互作用、牛顿定律
- **必修二**：曲线运动、平抛运动、圆周运动、万有引力、功和功率、机械能守恒
- **选修3-1**：电场、电路、磁场
- **选修3-2**：电磁感应
- **选修3-3**：热学
- **选修3-4**：振动与波、光学
- **选修3-5**：动量、原子物理
