# 物理王者 - 项目笔记

## 项目背景
为湖南高一学生（初中物理基础薄弱）开发的物理学习网页，融合《王者荣耀》游戏元素，目标是提升高考物理成绩。

## 用户需求
- 目标用户：湖南高一学生，初中物理基础弱
- 目标：查漏补缺，提升兴趣，备战高考
- 风格：活泼生动，融入王者元素
- 范围：初中+高中全部物理知识
- 重点：湖南高考真题练习

## 页面结构

### 1. 新手村 · 初中基础训练营（12英雄）
| 英雄 | 系别 | 核心内容 |
|------|------|---------|
| 疾跑·速度 | 基础系 | s=vt，平均速度测量 |
| 声波·音律 | 基础系 | 声音传播，三要素 |
| 光芒·反射 | 基础系 | 反射定律，透镜成像 |
| 冰火·物态 | 热学系 | 六种物态变化 |
| 天秤·密度 | 基础系 | ρ=m/V，天平使用 |
| 力量·基础 | 力学系 | 三种力，二力平衡 |
| 重压·压强 | 力学系 | P=F/S，液体/大气压强 |
| 浮力·阿基米德 | 力学系 | F浮=ρgV排，浮沉条件 |
| 杠杆·机械 | 力学系 | 杠杆平衡，机械效率 |
| 电流·欧姆 | 电学系 | I=U/R，串并联电路 |
| 功率·电能 | 电学系 | P=UI，焦耳定律 |
| 磁场·磁力 | 磁学系 | 电流磁效应，电机原理 |
