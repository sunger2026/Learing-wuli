圆周运动描述，线速度、角速度、周期、向心加速度、向心力。

📚

核心知识点

🎯 描述圆周运动的物理量
线速度：v = 2πr/T = ωr
角速度：ω = 2π/T = v/r
周期：T = 2π/ω = 2πr/v
转速：n = 1/T (单位：r/s)

关系：v = ωr（线速度 = 角速度 × 半径）

🔄 向心加速度
a = v²/r = ω²r = ωv = 4π²r/T²
  • 方向：指向圆心（时刻改变）
  • 物理意义：描述速度方向变化快慢
  • 匀速圆周运动加速度大小不变，方向时刻改变
⚡ 向心力
F = mv²/r = mω²r = mωv
  • 方向：指向圆心
  • 来源：可以是重力、弹力、摩擦力等或它们的合力
  • 效果：只改变速度方向，不改变速度大小
📐 传动装置特点
  • 同轴转动：角速度 ω 相同，v 与 r 成正比
  • 皮带传动：线速度 v 相同，ω 与 r 成反比
  • 齿轮传动：边缘线速度相同，方向相反

经典例题

📝 例题1：基本量计算

题目：

一物体做匀速圆周运动，半径 r = 0.5 m，周期 T = 2 s。求：

(1) 线速度 v

(2) 角速度 ω

(3) 向心加速度 a

解析：

(1) 线速度：

v = 2πr/T = 2π×0.5/2 = 0.5π ≈ 1.57 m/s

(2) 角速度：

ω = 2π/T = 2π/2 = π ≈ 3.14 rad/s

(3) 向心加速度：

a = v²/r = (0.5π)²/0.5 = 0.5π² ≈ 4.93 m/s²
📝 例题2：向心力来源

题目：

质量为 m 的小球在光滑水平面上做匀速圆周运动，绳子拉力为 F，半径为 r。若保持角速度不变，将半径增大到 2r，求新的拉力。

解析：

绳子拉力提供向心力：F = mω²r

当 r' = 2r，ω 不变时：

F' = mω²(2r) = 2mω²r = 2F

✅ 新拉力是原来的 2 倍

📝 例题3：圆锥摆

题目：

如图所示，质量为 m 的小球用长为 L 的细线悬挂，在水平面内做匀速圆周运动，细线与竖直方向夹角为 θ。求：

(1) 绳子拉力

(2) 小球运动的角速度

解析：

受力分析：重力 mg（向下），拉力 T（沿绳）

竖直方向：T·cosθ = mg

T = mg/cosθ

水平方向（向心力）：T·sinθ = mω²r

其中 r = L·sinθ（圆周运动半径）

代入：mg·tanθ = mω²L·sinθ

ω = √(g/Lcosθ)
