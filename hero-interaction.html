核心知识点

🌍重力

产生原因：地球对物体的吸引

大小：G = mg（m为质量，g为重力加速度）

方向：竖直向下（指向地心）

重心：重力的等效作用点

  • 质量分布均匀的物体：重心在几何中心
  • 重心可以在物体外部（如圆环）

⚠️ 重力 ≠ 地球引力（还有向心力影响）

🌀弹力

产生条件：

  • 物体直接接触
  • 发生弹性形变

方向：

  • 压力/支持力：垂直于接触面，指向被压/被支持物体
  • 绳子拉力：沿绳子指向绳子收缩方向
  • 弹簧弹力：沿弹簧轴线，指向原长位置

胡克定律：

F = kx

k为劲度系数，x为形变量（伸长或压缩量）

🛑摩擦力

产生条件：接触、挤压、粗糙、有相对运动或相对运动趋势

【滑动摩擦力】

f = μN
  • μ为动摩擦因数，与接触面材料有关
  • N为正压力（不一定等于重力！）
  • 方向：与相对运动方向相反

【静摩擦力】

  • 大小：0 < f静 ≤ fmax，由平衡条件求解
  • 最大静摩擦力：fmax = μ₀N（μ₀略大于μ）
  • 方向：与相对运动趋势方向相反

💡 口诀：滑压粗对动，静压粗对趋

📐受力分析步骤

步骤一：确定研究对象（隔离体）

步骤二：按顺序分析力

  1. 重力（一定存在，竖直向下）
  2. 弹力（看接触面，垂直于接触面）
  3. 摩擦力（看是否有相对运动/趋势）
  4. 其他力（如电场力、磁场力等）

步骤三：检查

  • 每个力都要有施力物体
  • 不分析内力（对整体受力分析时）
  • 检查是否漏力或多力
⚖️力的合成与分解

平行四边形定则：

两个共点力的合力，可以用表示这两个力的线段为邻边作平行四边形，对角线表示合力的大小和方向。

常用结论：

  • F₁、F₂夹角为θ，合力 F = √(F₁² + F₂² + 2F₁F₂cosθ)
  • F₁ = F₂ = F，夹角120°，合力 = F
  • 合力范围：|F₁ - F₂| ≤ F合 ≤ F₁ + F₂

正交分解法：将力沿x轴和y轴分解

Fx = Fcosθ，Fy = Fsinθ

  • 弹力：墙体碰撞，把你弹开
  • 摩擦力：减速带，让你停下来
  • 受力分析：团战前先看清敌方几个人在揍你😂

经典例题

📝例题1：摩擦力方向的判断

题目：如图，物体A放在传送带上，随传送带一起向上匀速运动。分析物体A受到的摩擦力方向。

传送带倾斜向上运行 →
┌─────────┐
│ 物体A │
└─────────┘

解析：

关键：判断相对运动趋势方向

假设传送带光滑（没有摩擦力）：

  • 物体A会由于重力沿斜面向下的分力而下滑
  • 即相对于传送带，物体有向下滑动的趋势

因此，静摩擦力的方向与相对运动趋势方向相反：

摩擦力方向沿斜面向上

✅ 答案：沿斜面向上

💡 技巧：假设光滑法判断摩擦力方向

📝例题2：弹簧连接体

题目：如图所示，质量为m的物体A放在水平地面上，通过弹簧（劲度系数为k）与墙壁连接。用手缓慢向右拉动物体A，当物体移动距离为x时，地面对物体A的摩擦力大小为多少？（动摩擦因数为μ）

解析：

分析物体A的受力：

  • 竖直方向：重力mg（向下），支持力N（向上），N = mg
  • 水平方向：弹簧拉力F = kx（向左，弹簧伸长x），摩擦力f（向右）

由于缓慢拉动，物体处于平衡状态：

f = F = kx（方向向右，与相对运动方向相反）

✅ 答案：摩擦力大小为kx，方向向右

💡 注意：此时摩擦力是滑动摩擦力，大小由平衡条件决定，不是μmg！

📝例题3：整体法与隔离法

题目：两个质量均为m的木块A、B叠放在水平地面上，A在上B在下。用水平力F拉A，使A、B一起匀速运动。求A对B的摩擦力和地面对B的摩擦力。

解析：

第一步：整体法分析

将A、B看作整体，匀速运动受力平衡：

  • 水平方向：拉力F向右，地面对B的摩擦力f地向左
  • f地 = F

第二步：隔离法分析B

物体B匀速运动，水平方向受力平衡：

  • 地面对B的摩擦力f地 = F（向左）
  • A对B的摩擦力fAB（向右，与f地平衡）
  • fAB = f地 = F

✅ 答案：A对B的摩擦力为F（向右），地面对B的摩擦力为F（向左）
