核心知识点
⚡ 动量与冲量

动量：描述物体运动状态的物理量

p = mv

冲量：力在时间上的积累效果

I = Ft
  • 动量是矢量，方向与速度方向相同
  • 冲量也是矢量，方向与力的方向相同
⚡ 动量定理

物体动量的变化等于所受合外力的冲量

Ft = mv₂ - mv₁ = Δp
🎮 王者技巧

动量定理是处理变力作用和短暂作用问题的利器！碰撞、打击、缓冲问题首选。

⚡ 动量守恒定律

条件：系统不受外力或所受合外力为零

m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
💡 注意：
1. 动量守恒是矢量式，需要规定正方向
2. 某一方向合外力为零，该方向动量守恒
3. 近似守恒：内力远大于外力时（如碰撞、爆炸）
💥 碰撞类型
🔄 弹性碰撞

动量守恒 + 动能守恒

特点：无能量损失，碰撞后分离

🔀 非弹性碰撞

动量守恒，动能有损失

特点：部分动能转化为内能

🎯 完全非弹性碰撞

动量守恒，碰撞后共速

特点：动能损失最大

v' = (m₁v₁ + m₂v₂)/(m₁ + m₂)
🚀 反冲运动

系统在内力作用下，一部分向某一方向运动，另一部分向相反方向运动。

典型应用：

  • 火箭发射（向后喷出燃气，火箭向前）
  • 开枪时的后坐力
  • 人船模型（人在船上走，船向后退）
0 = m₁v₁ + m₂v₂（系统初动量为零时）
💣 爆炸问题

爆炸过程特点：

  • 内力远大于外力，动量守恒
  • 化学能转化为动能，动能增加
  • 各部分向不同方向飞散
0 = m₁v₁ + m₂v₂（静止爆炸）

人船模型：m₁x₁ = m₂x₂（位移关系）

经典例题
📝 例题1：动量定理应用

一个质量为0.5kg的球以10m/s的速度水平飞来，被球拍击打后以15m/s的速度反向飞回。若球与球拍接触时间为0.02s，求球拍对球的平均作用力。

解：

设球飞回方向为正方向

初动量：p₁ = mv₁ = 0.5×(-10) = -5 kg·m/s

末动量：p₂ = mv₂ = 0.5×15 = 7.5 kg·m/s

根据动量定理：Ft = p₂ - p₁

F×0.02 = 7.5 - (-5) = 12.5

F = 12.5/0.02 = 625 N

方向与球飞回方向相同

📝 例题2：弹性碰撞

质量为m的小球A以速度v₀与静止的质量为2m的小球B发生弹性正碰，求碰撞后两球的速度。

解：

设碰撞后A球速度为v₁，B球速度为v₂，v₀方向为正

动量守恒：

mv₀ = mv₁ + 2mv₂ ...①

动能守恒：

½mv₀² = ½mv₁² + ½×2mv₂² ...②

由①：v₀ = v₁ + 2v₂

由②：v₀² = v₁² + 2v₂²

联立解得：

v₁ = -v₀/3（负号表示反向）

v₂ = 2v₀/3

💡 记忆口诀

弹性碰撞结论："交换速度"（质量相等时）；"一动一静，小撞大，小的反弹"

📝 例题3：人船模型

质量为M=200kg的船静止在水面上，船头到船尾距离L=5m。质量为m=50kg的人从船头走到船尾，不计水的阻力，求船移动的距离。

解：

系统初动量为零，水平方向动量守恒

设人相对地面位移为x₁，船相对地面位移为x₂

由动量守恒：0 = mv₁ + Mv₂（瞬时关系）

积分得：0 = mx₁ + Mx₂（位移关系）

又：x₁ - x₂ = L（人相对船的位移）

由0 = mx₁ + Mx₂，得 x₁ = -Mx₂/m

代入：-Mx₂/m - x₂ = L

x₂(-M/m - 1) = L

x₂ = -L·m/(M+m) = -5×50/250 = -1 m

船向后移动了1m（负号表示与人行走方向相反）

一般结论：x船 = -L·m/(M+m)
