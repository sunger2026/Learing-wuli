核心知识点

⚡ 欧姆定律（核心中的核心）
I = U / R

U = 电压（单位：伏特 V）

R = 电阻（单位：欧姆 Ω）

I = 电流（单位：安培 A）

💡 变形公式：

求电压：U = I × R

求电阻：R = U / I

📏 电学单位换算（必考！）

电流、电压、电阻的常用单位换算

物理量 基本单位 常用换算
电流 I 安培 (A) 1 A = 1000 mA
电压 U 伏特 (V) 1 kV = 1000 V
电阻 R 欧姆 (Ω) 1 MΩ = 1000 kΩ = 10⁶ Ω

🧮 记忆口诀："千进制，小变大除一千，大变小乘一千"

🔌 串联电路特点

定义：用电器逐个顺次连接，电流只有一条路径。

  • 电流处处相等：I = I₁ = I₂
  • 总电压等于各部分电压之和：U = U₁ + U₂
  • 总电阻等于各电阻之和：R = R₁ + R₂

💡 串联分压，电阻越大分得的电压越大

🔀 并联电路特点

定义：用电器并列连接在电路两点之间，电流有多条路径。

  • 干路电流等于各支路电流之和：I = I₁ + I₂
  • 各支路电压相等：U = U₁ = U₂
  • 总电阻倒数等于各电阻倒数之和：1/R = 1/R₁ + 1/R₂

💡 并联分流，电阻越小分得的电流越大

🧪 伏安法测电阻实验（中考必考！）

实验目的：用电流表和电压表测量未知电阻的阻值

实验原理：R = U/I（欧姆定律变形公式）

实验器材：

  • 电源（干电池或学生电源）
  • 待测电阻
  • 电流表（串联测电流）
  • 电压表（并联测电压）
  • 滑动变阻器（保护电路，改变电压电流）
  • 开关、导线若干

实验电路图：电流表串联，电压表并联在待测电阻两端

实验步骤：

  1. 按电路图连接电路，开关断开，滑片移到阻值最大处
  2. 闭合开关，调节滑动变阻器，读出电流表和电压表示数
  3. 改变滑动变阻器位置，测量3组不同的U、I值
  4. 根据 R = U/I 计算每次的电阻值
  5. 求平均值作为最终结果（减小误差）

⚠️ 注意：连接电路时开关必须断开；闭合开关前滑片要在阻值最大处！

实验结论：电阻是导体本身的性质，与电压、电流无关

王者峡谷里的电学

⚡ 用王者荣耀理解电流通路
  • 技能释放 = 电流通路：英雄释放技能需要蓝条（能量），就像灯泡发光需要电流通过
  • 蓝量 = 电压：蓝量越充足（电压越高），技能威力越大（电流越强）
  • 技能冷却 = 电阻：冷却时间长的技能（电阻大），单位时间内的释放次数少（电流小）
  • 团队配合 = 串并联：串联如配合连招，电流（伤害）处处相等；并联如分路推进，各路线独立进行

经典例题

📝 例题1：基础欧姆定律计算

题目：

一个电阻两端电压为 6V 时，通过的电流为 0.3A。

(1) 求这个电阻的阻值是多少？

(2) 当电压变为 12V 时，电流变为多少？

解析：

已知：U₁ = 6V，I₁ = 0.3A

(1) 求电阻：

根据欧姆定律 R = U/I

R = 6V / 0.3A = 20Ω

(2) 电压变化后：

电阻是导体本身的性质，不随电压电流改变，R = 20Ω

I₂ = U₂/R = 12V / 20Ω = 0.6A

✅ 答案：(1) 20Ω；(2) 0.6A

📝 例题2：串并联电路计算

题目：

两个电阻 R₁ = 10Ω，R₂ = 20Ω。

(1) 将它们串联接入电路，总电阻是多少？

(2) 将它们并联接入电路，总电阻是多少？

解析：

(1) 串联总电阻：

R = R₁ + R₂ = 10Ω + 20Ω = 30Ω

(2) 并联总电阻：

1/R = 1/R₁ + 1/R₂ = 1/10 + 1/20 = 3/20

R = 20/3 ≈ 6.67Ω

💡 记忆技巧：串联电阻越串越大，并联电阻越并越小！

📝 例题3：动态电路分析

题目：

如图所示电路，电源电压保持不变。当滑动变阻器的滑片向右移动时，电流表和电压表的示数如何变化？

（假设电路为定值电阻与滑动变阻器串联）

解析：

分析步骤：

① 滑片右移 → 滑动变阻器接入电阻变大

② 总电阻 R总 = R定 + R滑 变大

③ 电源电压 U 不变，根据 I = U/R，电流变小

④ 定值电阻两端电压 U定 = I×R定 变小

✅ 答案：电流表示数变小，电压表示数变小
