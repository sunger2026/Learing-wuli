核心知识点

⚡ 电功率公式（核心）
P = UI = I²R = U²/R

P = 电功率（单位：瓦特 W）

U = 电压（单位：伏特 V）

I = 电流（单位：安培 A）

💡 1kW = 1000W

🔥 焦耳定律
Q = I²Rt

Q = 热量（单位：焦耳 J）

I = 电流

R = 电阻

t = 时间

💡 电流越大、电阻越大、时间越长，产生的热量越多

💡 额定功率与实际功率

额定电压：用电器正常工作时的电压

额定功率：在额定电压下工作时的功率

实际电压：实际加在用电器两端的电压

实际功率：在实际电压下工作时的功率

💡 U实 = U额时，P实 = P额；U实 < U额时，P实 < P额

🏠 家庭电路与安全用电

电压：220V（交流电）

频率：50Hz

开关：接在火线上

保险丝/空气开关：电流过大时自动切断电路

金属外壳接地：防止触电
