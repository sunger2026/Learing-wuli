核心知识点

⚡ 电功率公式（核心）
P = UI = I²R = U²/R

P = 电功率（单位：瓦特 W）

U = 电压（单位：伏特 V）

I = 电流（单位：安培 A）

💡 1kW = 1000W

🔥 焦耳定律
Q = I²Rt

Q = 热量（单位：焦耳 J）

I = 电流

R = 电阻

t = 时间

💡 电流越大、电阻越大、时间越长，产生的热量越多

💡 额定功率与实际功率

额定电压：用电器正常工作时的电压

额定功率：在额定电压下工作时的功率

实际电压：实际加在用电器两端的电压

实际功率：在实际电压下工作时的功率

💡 U实 = U额时，P实 = P额；U实 < U额时，P实 < P额

🏠 家庭电路与安全用电

电压：220V（交流电）

频率：50Hz

开关：接在火线上

保险丝/空气开关：电流过大时自动切断电路

金属外壳接地：防止触电

王者峡谷里的电功率

⚡ 狂铁的能量系统 = 电功率

能量条 = 电能储存

狂铁的能量条就像电池，最多储存100点能量，代表电路中的电能储备。

💥 技能释放 = 电功率输出

一技能【碎裂之刃】

消耗：30能量 ⚡ P = W/t

就像电器工作，功率越大，单位时间消耗电能越多。狂铁的大招威力（功率）取决于当前能量值！

🔌 普攻充能 = 充电过程

每次普攻回复15能量

就像给电池充电，电流流入积累电能。充能速度 = 充电功率，充得越快，技能放得越频繁！

⏱️ 技能CD = 功率限制

冷却时间控制输出节奏

CD时间就像保险丝，防止功率过大烧毁电路。家庭电路中，保险丝会在电流过大时熔断，保护电器安全。

经典例题

例题1：电功率计算

题目：一个灯泡标有"220V 40W"字样，正常工作时通过灯丝的电流是多少？灯丝的电阻是多少？

解析：

已知：U = 220V，P = 40W

由 P = UI 得：I = P/U = 40W/220V ≈ 0.18A

由 P = U²/R 得：R = U²/P = (220V)²/40W = 48400/40 = 1210Ω

✓ 答案：电流约0.18A，电阻1210Ω

例题2：焦耳定律计算

题目：某电热水器的电阻为40Ω，通过的电流为5A，工作10分钟产生的热量是多少？

解析：

已知：R = 40Ω，I = 5A，t = 10min = 600s

由焦耳定律 Q = I²Rt

Q = (5A)² × 40Ω × 600s

Q = 25 × 40 × 600 = 600000J = 600kJ

✓ 答案：产生热量600000J（或600kJ）

王者峡谷里的电功率

⚡ 狂铁的能量系统

狂铁的技能释放就像电功率的应用：

  • 技能释放=消耗电能，功率越大伤害越高
  • 能量恢复=给电池充电（电能储存）
  • 技能CD=功率限制，不能无限释放

💡 功率P=UI，电压越大、电流越强，技能伤害越高！

🔋 蓝条与电能

英雄的蓝条就像电路中的电能：

  • 满蓝=额定电压，可以释放完整技能
  • 蓝量不足=实际电压低于额定，技能威力下降
  • 回蓝=电流做功，补充电能

💡 就像灯泡在电压不足时会变暗一样！

经典例题

📝 例题1：电功率计算

题目：

一个电热水器的额定电压为220V，额定功率为2000W。

(1) 求电热水器正常工作时的电流是多少？

(2) 如果实际电压只有200V，实际功率大约是多少？（假设电阻不变）

解析：

(1) 求电流：

根据 P = UI，得 I = P/U = 2000W / 220V ≈ 9.09A

(2) 求实际功率：

先求电阻：R = U²/P = (220V)² / 2000W = 24.2Ω

实际功率：P实 = U实²/R = (200V)² / 24.2Ω ≈ 1653W

💡 电压下降，功率下降，所以电热水器加热变慢！

📝 例题2：焦耳定律应用

题目：

一个电炉丝的电阻为50Ω，通过的电流为4A，通电10分钟。

(1) 求电炉丝产生的热量是多少焦耳？

(2) 如果电流增大到8A，相同时间内热量变为原来的多少倍？

解析：

(1) 求热量：

t = 10min = 600s

Q = I²Rt = (4A)² × 50Ω × 600s = 16 × 50 × 600 = 480000J

(2) 热量倍数：

Q与I²成正比，电流变为2倍，热量变为2² = 4倍

⚠️ 注意：电流过大容易烧坏电器，这就是保险丝的作用！
