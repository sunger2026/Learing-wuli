核心知识点

🚗 匀变速直线运动

定义：沿一条直线，且加速度不变的运动。

特点：

  • 加速度a恒定（大小和方向都不变）
  • 速度随时间均匀变化
  • 相等时间内速度变化量相等

💡 匀加速：a与v同向；匀减速：a与v反向

⚡ 加速度

定义：速度变化量与发生这一变化所用时间的比值。

a = Δv / Δt = (v - v₀) / t

单位：m/s²（米每二次方秒）

物理意义：描述速度变化快慢的物理量

⚠️ a > 0 不一定加速！要看a与v的方向关系

📐 五大运动学公式

基本公式（无x）：

v = v₀ + at

位移公式（无v）：

x = v₀t + ½at²

速度-位移公式（无t）：

v² - v₀² = 2ax

平均速度公式：

x = (v₀ + v)t / 2

中间时刻速度：

v(t/2) = (v₀ + v) / 2
📊 v-t 图像解读

纵轴：速度 v

横轴：时间 t

  • 斜率：加速度 a = Δv/Δt
  • 面积：位移 x（上正下负）
  • 截距：初速度 v₀

💡 斜率为正→加速；斜率为负→减速

📉 x-t 图像解读

纵轴：位移 x

横轴：时间 t

  • 斜率：速度 v = Δx/Δt
  • 截距：初始位置
  • 交点：两物体相遇

💡 x-t图中，曲线不是运动轨迹！

🎯 初速度为零的匀加速比例关系

设初速度 v₀ = 0：

  • 瞬时速度之比：v₁:v₂:v₃... = 1:2:3...
  • 总位移之比：x₁:x₂:x₃... = 1:4:9... (即 1²:2²:3²...)
  • 相等时间位移之比：xⅠ:xⅡ:xⅢ... = 1:3:5... (奇数比)
  • 相等位移时间之比：t₁:t₂:t₃... = 1:(√2-1):(√3-√2)...

⭐ 这些比例关系可以大大简化计算！

经典例题

📝 例题1：刹车问题

题目：

汽车以10m/s的速度行驶，刹车后获得的加速度大小为2m/s²，求：

(1) 刹车后3s末的速度；

(2) 刹车后6s内的位移。

解析：

第一步：求刹车停止时间

设初速度方向为正，则 v₀ = 10m/s，a = -2m/s²

由 v = v₀ + at，当 v = 0 时：

0 = 10 + (-2)t → t = 5s

⚠️ 刹车时间只有5s！之后汽车静止不动

(1) 刹车后3s末的速度：

v = v₀ + at = 10 + (-2)×3 = 4m/s

(2) 刹车后6s内的位移：

注意：6s > 5s，所以汽车在5s时已停止

只能用前5s计算，或用 v² - v₀² = 2ax

0² - 10² = 2×(-2)×x → x = 25m

✅ 答案：(1) 4m/s；(2) 25m

💡 易错点：直接用6s代入会得到错误答案！

📝 例题2：v-t图像分析

题目：

某物体的v-t图像如图所示（0-2s匀加速，2-4s匀速，4-6s匀减速至0）：

v(m/s)
4 | /‾‾‾‾\
| / \
0 |__/________\__ t(s)
0 2 4 6

(1) 求各段的加速度；

(2) 求6s内的总位移。

解析：

(1) 各段加速度：

  • 0-2s：a₁ = (4-0)/(2-0) = 2m/s²
  • 2-4s：a₂ = 0（匀速运动）
  • 4-6s：a₃ = (0-4)/(6-4) = -2m/s²

(2) 6s内的总位移（梯形面积）：

x = ½×(上底+下底)×高

x = ½×(2+6)×4 = 16m

或分段计算：x = ½×2×4 + 2×4 + ½×2×4 = 4 + 8 + 4 = 16m

✅ 答案：(1) a₁=2m/s², a₂=0, a₃=-2m/s²；(2) 16m

📝 例题3：追及问题

题目：

A、B两车在同一直线上同向行驶，A在前，速度为v₁=10m/s；B在后，速度为v₂=20m/s， 当B距A 50m时B开始刹车，加速度大小为2m/s²。问：B能否追上A？

解析：

关键思路：当两车速度相等时，若还未追上，则之后B速度小于A，更追不上。

第一步：求速度相等时的时间

v₁ = v₂ + at → 10 = 20 + (-2)t → t = 5s

第二步：计算此时两车的位移

A车位移：x₁ = v₁t = 10×5 = 50m

B车位移：x₂ = v₂t + ½at² = 20×5 + ½×(-2)×5² = 100 - 25 = 75m

第三步：判断是否追上

B比A多走的距离：75 - 50 = 25m

初始距离：50m

25m < 50m，所以追不上

最小距离：50 - 25 = 25m

✅ 答案：B不能追上A，最小距离为25m
